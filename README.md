# doc-mind
Offline Streamlit RAG app that turns any PDF into an interactive study tool: summary, questions, chat &amp; quiz. Fully private &amp; local using Ollama. Tech: Streamlit • LangChain • Ollama (qwen2.5:3b) • FAISS • HuggingFace Embeddings • PyPDF Perfect for students &amp; AI engineers exploring local RAG pipelines.
